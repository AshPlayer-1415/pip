# Pip Screenshot Guide

Capture screenshots on macOS in dark mode. Use a clean desktop background, hide unrelated menu bar items when possible, and crop tightly around Pip unless the floating bubble context is important.

Save final screenshots in:

```text
marketing/screenshots/
```

Recommended filenames:

```text
01-onboarding-welcome.png
02-personality-picker.png
03-today-dashboard.png
04-reminder-creation.png
05-private-mode.png
06-presentation-safe-mode.png
```

## 1. Onboarding

Capture the first welcome screen after a fresh Reset Pip or clean first launch.

Show:

- Pip welcome copy
- companion avatar
- compact dark panel
- no personal reminder data

Suggested filename: `01-onboarding-welcome.png`

## 2. Avatar / Personality Picker

Capture the onboarding personality selection step.

Show:

- all personality options
- selected personality state
- visually distinct CSS avatars
- calm, premium dark theme

Suggested filename: `02-personality-picker.png`

## 3. Today Dashboard

Capture the main dashboard after onboarding.

Show:

- Today section
- active mode
- next reminder
- missed count
- quick actions
- companion identity in the header

Suggested filename: `03-today-dashboard.png`

## 4. Reminder Creation

Capture the Add Reminder view.

Show:

- title field
- time field
- type picker
- Save Reminder button

Use neutral sample text such as `Check plants`.

Suggested filename: `04-reminder-creation.png`

## 5. Private Mode

Capture the dashboard or settings with Private Mode enabled.

Show:

- Private active state
- generic/private reminder text where visible
- no sensitive reminder details in the screenshot

Suggested filename: `05-private-mode.png`

## 6. Presentation Safe Mode

Capture the dashboard with Presentation Safe Mode enabled.

Show:

- Safe Mode active state
- Presentation Safe mode label
- queued or missed quiet-mode context if available

Suggested filename: `06-presentation-safe-mode.png`
